- 低饱和度配色，护眼舒适

- 移动端优先设计

## 技术栈说明

- **Vue 3** - 渐进式 JavaScript 框架
- **Vite** - 下一代前端构建工具
- **Pinia** - Vue 3 官方状态管理库
- **Vue Router** - Vue.js 官方路由
- **Element Plus** - Vue 3 组件库
- **SCSS** - CSS 预处理器
- **Axios** - HTTP 客户端（预留）
- **dayjs** - 轻量级日期库
- **localStorage** - 浏览器本地存储
