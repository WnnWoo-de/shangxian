# 废品/布碎采购管理系统 - 详细实施计划

## 项目概述
- **技术栈**：Vue3 + Vite + Pinia + Vue Router + SCSS + Axios
- **架构**：纯前端，数据存储在 localStorage/indexedDB
- **设计风格**：浅色调主题
- **适配**：Web端 + 移动端响应式

## 六、技术要点

### 6.1 响应式设计
- 移动端优先设计
- 断点：360px, 480px, 768px, 1024px, 1280px
- 使用 CSS Grid + Flexbox
- 触摸友好的交互

### 6.2 数据持久化
- 使用 localStorage 存储基础数据
- 使用 indexedDB 存储大量单据数据
- Pinia 状态持久化

### 6.3 性能优化
- 路由懒加载
- 虚拟列表（长列表）
- 防抖/节流
- 缓存策略

### 6.4 草稿机制
- 自动保存（输入后3秒）
- 路由离开时提示保存
- 页面关闭前保存
- 进入页面时检测并提示恢复
